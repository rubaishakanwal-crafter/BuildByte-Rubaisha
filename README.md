
"SkillBridge"
It is a student career guidance platform designed to help learners track skills, explore career paths, and access personalized learning resources.

Features:
Student Dashboard – Overview of progress and activities

Career Passport – Personalized career roadmap

Skill Tracker – Monitor skill development

Learning Portal – Curated learning materials

Project Recommendations – Suggested projects based on skills

Career Quiz – Interactive quiz for career insights

User Authentication – Secure login and signup system

Technologies Used:
HTML5
CSS3
JavaScript
